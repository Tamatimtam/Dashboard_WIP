About This Project

Project Overview
GenZ Financial Profile Dashboard

This Flask web application provides an interactive analysis of GenZ financial behavior, focusing on income and expense patterns. The dashboard visualizes the relationship between income categories and expense categories using a Diverging Bar Chart.

Key Features
  • Data Summary: Comprehensive overview of dataset statistics
  • Interactive Visualization: Diverging Bar Chart powered by Plotly
  • Financial Insights: Key metrics and patterns in GenZ finances
  • API Endpoints: JSON API for programmatic access
  • Responsive Design: Bootstrap-powered mobile-friendly interface
Dataset Columns
Column Description
avg_monthly_income Average monthly income (numerical)
avg_income_category Income category (e.g., <2jt, 2-4jt, etc.)
avg_monthly_expense Average monthly expense (numerical)
avg_expense_category Expense category (e.g., <2jt, 2-4jt, etc.)
financial_standing Financial status (Surplus, Deficit, Break-even)
